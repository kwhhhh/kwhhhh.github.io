Latest tool to implement stagger DID, fect(Yiqing Xu)
Use method “ife”:

fect pm25_per_light,treat(reform) unit(dist_code) time(year) method(“ife”)

placebo test

fect pm25_per_light, treat(reform) unit(dist_code) time(year) method(“ife”) se r(2) placeboTest

wald test

fect pm25_per_light, treat(reform) unit(dist_code) time(year) method(“ife”) se r(2) preperiod(-14) offperiod(5) wald nboots(100)

equitest

fect pm25_per_light, treat(reform) unit(dist_code) time(year) method(“ife”) se r(2) preperiod(-14) offperiod(5) equiTest

For what the tests above mean, please refer Yiqing Xu
